The Math Behind ML

Linear Regression — the simplest predictive model.

ŷ = wx + b
w

Weight — slope of the line. Controls how much input affects output.

b

Bias — y-intercept. Shifts the prediction up or down.

ŷ

Prediction — the output value the model estimates for a given x.

The model finds optimal w and b by minimizing the Mean Squared Error (MSE) between predictions and actual values using gradient descent.

Code Example

A simple ML prediction in Python using scikit-learn.

predict.py 
from sklearn.linear_model import LinearRegression
import numpy as np

# Training data: hours studied → exam score
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([20, 40, 60, 75, 90])

# Train the model
model = LinearRegression()
model.fit(X, y)

# Predict score for 6 hours of study
prediction = model.predict([[6]])
print(f"Predicted score: {prediction[0]:.1f}")
# Output: Predicted score: 103.0

Advantages

⚡ Automation

Handles repetitive tasks 24/7 without fatigue, freeing humans for creative work.

🎯 Accuracy

Processes millions of data points to make precise predictions and decisions.

🚀 Speed

Analyzes data in seconds that would take humans days or weeks.

Limitations

⚖️ Bias

Models can inherit and amplify biases present in their training data.

💼 Job Displacement

Automation may replace certain roles, requiring workforce reskilling.

🔒 Privacy

Large-scale data collection raises serious surveillance and privacy concerns.
